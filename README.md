# ATS-Resume-Screening-System
This project is a Python-based Application Tracking System (ATS) that automates resume screening by matching candidate resumes against a job description.  
It provides a user-friendly Python script to upload files, analyze resumes, and filter top candidates based on keyword matching.

Supports multiple file formats:
1. PDF
2. DOCX
3. TXT

Keyword-based resume matching

Match percentage calculation

Automatically filters and copies selected resumes

Interactive GUI using Tkinter

Multithreading for smooth UI performance

Real-time progress tracking
